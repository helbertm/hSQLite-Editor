Ajuda Geral do hSQLite Editor

Ferramenta para análise de bases SQLite com foco em suporte/TI e uso didático em aulas de banco de dados.

1. Começando

Use Escolher .db para abrir uma base. Depois escreva SQL e execute com F5.
Extensões aceitas para bancos SQLite: .db, .sqlite, .sqlite3, .db3, .s3db, .sl3, .sqlite.db.

2. Área SQL

Abas independentes, histórico, favoritos, abrir/salvar arquivo SQL e atalhos para fluxo rápido.

3. Schema

Filtre objetos por nome e por tipo. Clique em colunas para inserir objeto.coluna no editor.

4. Resultados

Ordenação multi-coluna, paginação, seleção de linhas, exportação CSV/JSON e cabeçalho sempre visível.

5. Persistência

Sessão, histórico, favoritos e preferências ficam no navegador (localStorage/IndexedDB).

6. Modo offline

Para uso offline avançado, veja a orientação na tela de conexão/offline e prefira abrir via http://localhost.

7. Reporte de bug

Use o botão Reportar Erros para abrir o GitHub Issues do projeto.
